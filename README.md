# Course Advisor AI

An AI-powered system that helps students manage course timelines and excel academically by answering course policy related questions, reminding students of deadlines, and suggesting strategies to improve grade.

## Features
- Automatic syllabus and schedule parsing from PDFs
- Natural language Q&A about course policies and schedule
- Grade tracking, predictions, and natural language suggestions to hit goals
- Deadline reminders

## Technologies
- **Document Processing**: PyMuPDF, pdfplumber
- **LLMs**: OpenAI GPT-4
- **NLP**: sentence-transformers, FAISS
- **Backend**: FastAPI
- **Frontend**: Lovable
