# Setup Instructions

## Prerequisites
- Python 3.9+
- pip
- Git

## Installation Steps

1. Clone repository
2. Create virtual environment: `python3 -m venv venv`
3. Activate: `source venv/bin/activate`
4. Install: `pip install -r requirements.txt`
5. Create `.env` file with API keys
6. Run test: `python test_setup.py`
